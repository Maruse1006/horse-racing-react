# Welcome to your Expo app 👋

ナミュールのデータを例にして、generation_1（例えば「ハービンジャー」）が8行続くケース
```
spanMap = {};
count = 0;

// ループ開始
i = 0;  // 最初の行
count = 1;  // 初回なのでカウントを1に設定

// iが進むごとに値が同じであれば
i = 1; count = 2;
i = 2; count = 3;
// ...

// 最後の行 or 異なる値に到達した時
i = 8;  // 現在の行が異なる値
if (count > 0) {
    spanMap[0] = 8;  // 最初の行に行数（8行）を割り当て
}
```

spanMapの内容は以下のようになります。
```
spanMap = {
    0: 8,  // 0行目から始まるグループは8行連続
};
```
